About

Blanket is a RESTy wrapper for long-running tasks. Define task types as TOML files, submit them via REST API or CLI, and let workers execute them — all from a single binary with this web UI baked in.

See the project on GitHub for documentation, releases, and source. Currently registered workers are listed on the Workers page.
